## Fonts

To use a custom font like **Kugile** or **Orange Avenue**, download the font file (TTF/OTF/WOFF2) and place it in this folder.

Recommended filenames:
- `Kugile.woff2` / `Kugile.ttf`
- `OrangeAvenue.woff2` / `OrangeAvenue.ttf`

Example source page (Orange Avenue): `https://www.1001fonts.com/elegant-fonts.html`
